Log a set, land on the partner
Tapping "Log set" records the set and immediately swaps the screen to the partner exercise, prefilled with that exercise's own draft numbers (the flow navigator already replaces screens with no animation, so the swap is instant). The partner chip under the title hops over without logging, saving an edit to an old set never swaps, and "Next exercise" keeps its normal meaning: leave the superset and move on. Building supersets into routines
In Edit Routine each row gains the same native ellipsis menu as the exercise screen (the RoutineList pattern, with an Alert fallback off iOS) offering "Superset with above". Grouped rows share one bordered container with the SUPERSET overline and drag as a unit. Saving sends the new optional exercises array with supersetGroup; starting the routine hydrates the groups into the live workout.
