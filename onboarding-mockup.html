Gear · Onboarding — 34-screen design mockup

Static visual reference for the flow in frontend/src/navigation/onboarding/. Non-functional — for reviewing and editing the design. Tweak the palette/spacing in the :root block at the top of this file and every screen updates.

33 ordered steps (0–32) from steps.ts, plus the Trial-boost overlay that surfaces from the paywall — 34 phone cells. The tester-only Skip / Back dev pills are intentionally omitted; only real UI affordances (the top-bar back chevron, "Skip for now", "Maybe later", "Sign in", etc.) are shown. Screens are rendered in a representative selected / filled state.
