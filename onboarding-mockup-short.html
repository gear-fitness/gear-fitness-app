Gear · Onboarding — short version (15 screens)

Static visual reference for the flow in frontend/src/navigation/onboarding/, for reviewing and editing the design. Tweak the palette/spacing in the :root block at the top of this file and every screen updates. The gear videos and logo are the real app assets, embedded as data URIs, so the file is fully self-contained.

A trimmed 15-step variant of the full flow (see onboarding-mockup.html): Welcome, Name, Sex, Birthday, Height, Weight, Activity, Goal weight, Progress projection, Apple Health, Notifications, Username, Profile photo, Create account, Paywall. Progress bars are recomputed for the shorter flow. Screens are rendered in a representative selected / filled state; paywall prices are representative (the app pulls live prices from RevenueCat).
