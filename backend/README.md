# This is our backend

## How to run

1. Set up environment variables

```bash
cp run.sh.template run.sh
```

Edit `run.sh` to contain correct values.

2. Make it executable

```bash
chmod +x run.sh
```

3. Start docker

```bash
docker-compose up -d
```

4. Run the application

```bash
./run.sh
```
